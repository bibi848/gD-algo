# gD-algo

Welcome to gD-algo! I use this library for financial programming projects. Currently, I am exploring historical stock and crypto data, but I also have projects looking at option pricing and using the polymarket API.

## Summary
Folders:
* Classes: Any new objects defined live in this folder.
* Historical Data: Contains Stock and Crypto historical data. More information can be found in the folder.
* Option Tests: Scripts looking at option pricing.
* PM: Scripts looking at using Polymarket via its API.
<br>
Files:
* `.gitignore`
* README.md
* requirements.txt
* test bench.py
